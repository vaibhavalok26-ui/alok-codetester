# Alok CodeTester

**Alok CodeTester** is an interactive online code editor inspired by CodePen. It allows users to write and test **HTML, CSS, and JavaScript** code live in the browser. The website is designed for beginners and developers to experiment with code, learn, and see results instantly.

---

## Features

- **HTML, CSS, and JS Editors:** Separate editors for each code type with syntax highlighting using **CodeMirror**.  
- **Live Output Panel:** Run your code and see live results instantly in an iframe.  
- **Dark/Light Mode:** Toggle between dark and light themes for comfortable coding.  
- **Resizable Panels:** Adjust the size of editor and output panels dynamically.  
- **Demo / Example Feature:** Automatically loads demo code (‚ÄúNamaste‚Äù example) to show how it works.  
- **Sign-Up Button:** Placeholder for future signup and user registration functionality.  
- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.

---

## How to Use

1. Write your **HTML, CSS, and JS** code in the respective editors.  
2. Click **Run Code** to see the live output below.  
3. Use the **Dark/Light toggle** button to switch themes.  
4. Click **Show Example** to automatically load a sample ‚ÄúNamaste‚Äù code.  
5. (Future) Use the **Sign Up** button to register and save your code snippets.
